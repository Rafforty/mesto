# Проект: Место

### Обзор
Учебный проект в рамках программы Яндекс.Практикума "Веб-разработчик".

**При выполнении работы были использованы данные технологии**
1. Flexbox верстка
2. БЭМ 
3. Адаптивная верстка
4. DOM (Document Object Model)
5. JS скрипты
6. ООП
7. Классы
